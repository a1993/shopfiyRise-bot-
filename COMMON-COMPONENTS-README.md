# 通用组件快速参考

## ✅ 已完成的通用组件（可作为分区直接使用）

### 1. **标题组件** - `sections/common-heading.liquid`
- 可设置 H1-H6 标签
- 主标题 + 副标题
- 4种尺寸（小/中/大/超大）
- 响应式字体大小（PC/M不同）
- 背景颜色支持

**在后台添加**: 主题编辑器 → 添加分区 → 搜索 "标题 Heading"

---

### 2. **Banner 组件** - `sections/common-banner.liquid`
- PC/Mobile 分别设置图片
- 4种宽度模式（全屏/标准1368px/宽屏1600px/窄屏1024px）
- 移动端自动全宽
- 可添加链接

**在后台添加**: 主题编辑器 → 添加分区 → 搜索 "横幅 Banner"

---

### 3. **内容区域组件** - `sections/common-content-area.liquid`
- 可放置任何内容的通用容器
- 支持 5 种 Block：
  - 文本块
  - 富文本块
  - 图片块
  - 按钮块
  - 间距块
- 背景颜色/渐变支持

**在后台添加**: 主题编辑器 → 添加分区 → 搜索 "内容区域 Content"

---

### 4. **容器组件** - `snippets/common-container.liquid` ⚠️
- Snippet，不能直接添加，供其他组件内部使用
- 统一管理页面宽度（标准1368px）

---

### 5. **Modal 组件** - `snippets/common-modal.liquid` ⚠️
- Snippet，通过代码调用
- 用于视频、图片等弹窗
- 5种尺寸（小/中/大/超大/全屏）

---

## 🎯 标准规范

### 容器宽度
- **标准**: 1368px (85.5rem) - **推荐活动页使用**
- **宽屏**: 1600px (100rem)
- **窄屏**: 1024px (64rem)
- **全宽**: 100%

### 间距大小
| 大小 | 移动端 | 桌面端 |
|-----|-------|-------|
| 小 | 16px | 16px |
| 中 | 24px | 32px |
| 大 | 40px | 64px |

### 响应式断点
- **移动端**: ≤ 767px
- **平板端**: 768px - 1279px
- **桌面端**: ≥ 1280px
